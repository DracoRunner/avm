avm is Any Version Manager by Prajanova.

Package:
- npm: @prajanova/avm
- GitHub: prajanova/avm

Use:
- configure project commands in `.avm.json`
- use `aliases` for commands
- use `env` for environment variables
- use `tools` for selected tool versions
- run `avm shims install` for plain binary interception
- run `eval "$(avm shell-init)"` in shell setup

Important:
- local `.avm.json` overrides global `~/.avm.json`
- no automatic tool install during execution
- missing managed Node version falls back to system Node with warning
- Node package scripts are discovered from `package.json`

Commands you can use:
- `avm init`
- `avm add [--global] <alias> <command>`
- `avm remove [--global] <alias>`
- `avm list`
- `avm which <alias-or-tool>`
- `avm run <alias> [args...]`
- `avm resolve <alias> [args...]`
- `avm env`
- `avm tool use [--global] <tool> <version>`
- `avm tool install <tool> <version>` (reserved provider surface; Node download is not implemented in the baseline)
- `avm tool uninstall <tool> <version>`
- `avm tool list`
- `avm plugin add|list|remove|update`
- `avm shims install|remove|path`
- `avm shell-init`
- `avm version`
